# Qit WebUI Frontend

This frontend is the React and Tailwind surface for the Qit web session.

## Local development

```bash
npm install
npm run dev
```

## UI preview

Use the lightweight component review surface while refining tokens and primitives:

```text
http://localhost:5173/?preview=ui
```

The preview route is intentionally small. It exists to review shared UI building blocks without navigating the full product flow.
