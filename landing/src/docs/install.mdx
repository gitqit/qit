# Install Qit

Qit is designed to be fast to try: install it in one command, point it at a folder, and get a temporary authenticated Git remote plus a local Web UI.

## Install with Homebrew

If you use Homebrew on macOS or Linux, this is the fastest path:

```bash
brew tap gitqit/qit
brew install qit
```

If you prefer a single command, Homebrew can install directly from the tap:

```bash
brew install gitqit/qit/qit
```

Confirm the binary is available:

```bash
qit --help
```

## Download a release

If you do not use Homebrew, install from the [Qit releases](https://github.com/gitqit/qit/releases) page instead.

1. Download the archive for your platform.
2. Extract the `qit` binary somewhere on your `PATH`.
3. Confirm the binary is available with `qit --help`.

## Build from source

If you prefer to build locally, clone the repo and compile the CLI:

```bash
git clone https://github.com/gitqit/qit.git
cd qit
cargo build --release
./target/release/qit --help
```

## What you need

- A normal project folder you want to share.
- Git installed on the host machine.
- A recent browser if you want to use the local Web UI.
- An ngrok token if you want Qit's default public transport out of the box.
